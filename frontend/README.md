# Frontend - Correos Masivos

Aplicación frontend construida con React, TypeScript y Vite.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Estructura del Proyecto

```
src/
├── components/     # Componentes reutilizables
├── pages/         # Páginas de la aplicación
├── services/      # Servicios API
├── hooks/         # Custom hooks
├── context/       # Context providers
├── types/         # TypeScript types
└── utils/         # Utilidades
```

## Tecnologías

- React 18
- TypeScript
- Vite
- React Router
- Axios
